# Инструкция по работе с сайтом пекарни

---

## Для клиентов — как оформить заказ

Откройте сайт пекарни в браузере.

Заполните форму:

1. **Имя** — как вас зовут, чтобы пекарь знал чей заказ.

2. **Контакты** — заполните хотя бы одно поле: телефон, Telegram, WhatsApp или MAX. Если введёте телефон, он должен содержать не менее 10 цифр — иначе форма не отправится.

3. **Дата получения** — нажмите на поле и выберите день. После выбора появится подсказка:
   - 🟢 **«Свежая выпечка на этот день 🍞»** — и список того, что будет в печи. Значит, заказ примут.
   - 🟡 **«😔 В этот день пекарня не работает 🍞»** — в этот день заказ оформить нельзя, выберите другую дату.

4. **Что заказываете** — напишите, что хотите и сколько штук. Например: *ржаной 2 шт, белый батон 1 шт*.

5. **Способ получения** — выберите: в пекарне или в подъезде.

6. Нажмите **«Отправить заказ»**. На экране появится подтверждение.

## Для администратора — панель управления

Перейдите по адресу вашего сайта с добавлением `/admin`.

- Логин и пароль заданы в файле `.env` на сервере (`ADMIN_LOGIN`, `ADMIN_PASSWORD`).

Сессия активна 8 часов, после этого нужно войти снова.

### Расписание выпечки

Здесь вы заранее указываете, что будет выпекаться в конкретный день. Клиенты увидят эту информацию при выборе даты в форме заказа.

**Как добавить день:**
1. В поле «Дата» выберите нужный день.
2. В текстовом поле напишите, что будете печь. Каждый вид хлеба — с новой строки. Например:
   ```
   Ржаной хлеб на закваске
   Белый батон
   Булочки с корицей
   ```
3. Нажмите **«Сохранить»**.

Если выбрать дату, для которой запись уже есть — текст загрузится автоматически, можно отредактировать и сохранить заново.

**Список записей** отсортирован по дате — ближайшие сверху.

**Кнопка «Скрыть»** — убирает запись из списка, но не удаляет. Удобно для прошедших дней, чтобы не мешались. Скрытые записи можно посмотреть кнопкой **«Показать скрытые»** вверху раздела.

**Кнопка «Удалить»** — удаляет запись. Перед удалением появится вопрос-подтверждение. Запись исчезнет из списка навсегда (технически сохраняется в базе для резервной копии, но вы её не увидите).

### Заказы

Здесь вы видите все заказы на выбранный день.

**Как посмотреть заказы:**
1. В поле «Дата» выберите нужный день.
2. Нажмите **«Показать»**.
3. Появится список заказов: имя клиента, контакты, что заказал, способ получения и время оформления.

По умолчанию выданные заказы скрыты, чтобы не отвлекали. Кнопка **«Показать выданные»** вверху списка раскроет их.

**Как отметить заказ выданным:**
- Нажмите **«Выдан»** на карточке заказа — карточка приглушится, статус сменится на зелёный «✓ Выдан».
- Если отметили по ошибке — нажмите **«Отменить»**, статус вернётся к «К выдаче».

**Кнопка «Удалить»** — убирает заказ из списка (например, если клиент отменил). Перед удалением появится вопрос-подтверждение. Кнопка намеренно расположена отдельно от «Выдан», чтобы не нажать случайно.

### Пример рабочего дня

**С вечера или утром:**
1. Открыть панель управления.
2. В разделе «Расписание выпечки» добавить запись на завтра — написать что будет в печи.

**В течение дня:**
1. Открыть раздел «Заказы», выбрать сегодняшнюю дату.
2. Видите список — кто и что заказал, как заберёт.
3. По мере выдачи нажимать «Выдан» на каждом заказе.

**Если клиент передумал:**
- Найти его заказ и нажать «Удалить».
